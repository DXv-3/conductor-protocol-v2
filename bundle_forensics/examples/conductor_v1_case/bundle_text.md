# Conductor v1 Bundle (Example Case)

This is a reference artifact used for forensic demonstration.

Claims present in this bundle:
- Cost cap enforced per session (configurable, default $5.00).
- Proof gate blocks promotion without runtime evidence.
- Memory artifact persists trace across resets.
- Just listen background cycle collects observations passively.
- Config values loaded from YAML at startup.
- Build on top only permitted after canonical gate pass.
